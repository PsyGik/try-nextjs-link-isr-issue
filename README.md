## Running locally
- `npm install`
- `npm run build`
- `npm run start`

- Navigate to `localhost:3000` on the browser. 
- Click on `About` under `Using next/link`
- Open Network console and observe that `about.json` is fetched
- In your local terminal observe the logs for `getStaticProps`
